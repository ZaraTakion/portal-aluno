# Portal do Aluno

## 📌 Descrição
Sistema web completo para gestão acadêmica, permitindo acesso a informações por alunos, professores e responsáveis.

## 🎯 Objetivo
Modernizar e melhorar a experiência do portal acadêmico existente, com foco em usabilidade, organização e performance.

## 👥 Usuários
- Aluno
- Professor
- Responsável

## 🧱 Tecnologias
- Frontend: React
- Backend: Django
- Banco de dados: PostgreSQL

## ⚙️ Funcionalidades principais
- Login com autenticação
- Visualização de notas
- Controle de frequência
- Área financeira
- Perfil do usuário

## 📂 Estrutura do projeto
- /frontend → Interface
- /backend → API e regras de negócio
- /docs → documentação
