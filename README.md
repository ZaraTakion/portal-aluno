# Portal Acadêmico Full-Stack

## Objetivo
Desenvolver uma plataforma completa de portal acadêmico para alunos, professores e responsáveis.

## Usuários
- Aluno
- Professor
- Responsável

## Módulos

### Autenticação
- Login com RA
- Controle de acesso (RBAC)

### Acadêmico
- Notas
- Frequência

### Financeiro
- Mensalidades
- Status

### Perfil
- Dados do usuário

### Professor
- Lançar notas
- Lançar faltas

### Responsável
- Acompanhar aluno

## Arquitetura

Frontend: React  
Backend: Django REST  
Banco: PostgreSQL  
Auth: JWT  

## Estratégia

### Fase 1 (MVP)
- Login
- Dashboard
- Notas
- Frequência

### Fase 2
- Financeiro
- Perfil

### Fase 3
- Professor
- Responsável
