# Portal Acadêmico

## 📌 Descrição
Sistema web completo para gestão acadêmica de uma instituição de ensino superior.

## 🎯 Objetivo
Criar uma plataforma moderna, funcional e intuitiva para alunos, professores e responsáveis.

## 👥 Usuários
- Aluno
- Professor
- Responsável

## 🧱 Stack
- Frontend: React
- Backend: Django
- Banco: PostgreSQL

## 🚧 Status
Projeto em fase inicial (estruturação)
