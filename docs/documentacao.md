# Portal do Aluno

## Objetivo
Reestruturar o portal acadêmico para torná-lo mais intuitivo, funcional e organizado.

## Problema
Sistema atual confuso, lento e difícil de usar.

## Usuários
- Aluno
- Professor
- Responsável

## MVP
- Login
- Dashboard
- Notas
- Frequência
- Financeiro
- Perfil

## Fluxo principal
1. Login
2. Dashboard
3. Escolher:
   - Notas
   - Frequência
   - Financeiro

## Stack
- Frontend: React
- Backend: Django
- Banco: PostgreSQL
- Auth: JWT
