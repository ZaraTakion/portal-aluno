# Arquitetura do Sistema

## Modelo

Frontend (React) → Backend (Django REST) → Banco (PostgreSQL)

## Autenticação
JWT com RBAC (controle por tipo de usuário)

## Papéis do sistema
- Aluno
- Professor
- Responsável

## Módulos principais
- Autenticação
- Acadêmico (notas, frequência)
- Financeiro
- Perfil
